Basically the program, zooms in on the center of the monitor using an OS thing (https://learn.microsoft.com/en-us/windows/win32/api/magnification/nf-magnification-magsetfullscreentransform).The user can set up settings so they can use muliple keys to zoom in at different multipliers, and can also save these settings. The real part of the code that I care about is the UI. Later Ill do more things because Im interested but for now Ill probably trim it for the CPT. The goal of the program is to make it easier to inspect small on-screen details and improve accessibility for users who need larger text or visuals. It is a simple, customizable, open source program, and the default windows magnifier has multi-key hotkeys, (slow, clunky, and non customizable), and it's isn't stationary (it moves with the mouse which can be annoying). Hopefully I will be able to expand the program to be able to use mouse keys as well. Most articles and things don't go all the way to the edges of the screen, which is why the program zooms in on the center.
